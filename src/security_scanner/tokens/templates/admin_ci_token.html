Security notes

Storage
Only the SHA-256 hash of the token is stored. The plaintext is shown once at generation time and never persisted.
Revocation
Rotating immediately marks the old token as revoked. Any in-flight CI request using the old token that hasn't reached the server yet will 401. There is no grace period.
Format
phs_ci_<32-bytes-url-safe-base64>. About 256 bits of entropy, collision-infeasible.
Validation
The CI handler computes sha256(provided_token) and uses hmac.compare_digest against the stored hash for constant-time comparison, preventing timing attacks.
