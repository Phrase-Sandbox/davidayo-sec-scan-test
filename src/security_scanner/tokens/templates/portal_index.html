How to use it

Pass the token as a bearer header when calling /scan/local:

curl -H "Authorization: Bearer phs_local_tok-..." \
     -d '{"files": {...}}' \
     https://<scanner>/scan/local

Or run phrase-sec-scan login from your machine — it will open this portal in your browser and write the token to ~/.phrase-sec-scan/config.yaml.
