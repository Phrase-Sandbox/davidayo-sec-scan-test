{# ── Security notes ── #}

Security notes

Storage
Only the SHA-256 hash of the token is stored. The plaintext is shown once at generation and never persisted.
Revocation
Rotating immediately marks the old token revoked. No grace period — in-flight requests with the old token will 401.
Format
phs_ci_<32-bytes-url-safe-base64> — ~256 bits of entropy.
Validation
Uses hmac.compare_digest — constant-time comparison prevents timing attacks.
